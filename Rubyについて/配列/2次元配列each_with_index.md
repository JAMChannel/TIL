# 2次元配列で地図を表示する

```
landmap = Array.new(10).map{Array.new(20,"森")}
landmap[0][0] = "城"
landmap[0][19] = "町"
landmap[9][19] = "町"
landmap.each_with_index do |line,i|
    print "#{i + 1}"
    line.each do |area|
        print area
    end
    puts " "
end
```
