```
$ zip -e [圧縮後のファイル名] [圧縮するファイル名]
Enter password: 設定するパスワード
Verify password: パスワードを再入力
```

## file名
```
$ zip -e [圧縮後のファイル名] [圧縮するファイル名]
Enter password: 設定するパスワード
Verify password: パスワードを再入力
```
```
$ zip -e sample.zip sample.csv
```

 ## zip -erとすることでディレクトリを圧縮することも可能
 ```
$ zip -er [圧縮後のファイル名] [圧縮するディレクトリ名]
Enter password: 設定するパスワード
Verify password: パスワードを再入力
 ```

```
$ zip -er sample.zip sample
```
